# NGX Brand Voice Skill

Esta skill proporciona el contexto completo de la voz de marca NGX para generar contenido alineado.

## Cuándo Usar Esta Skill

Activa esta skill cuando necesites:
- Escribir cualquier tipo de copy para NGX
- Adaptar contenido a audiencias específicas
- Verificar si un texto cumple con la voz de marca
- Generar ejemplos de los diferentes modos de comunicación

## Archivos Incluidos

- `voice-guide.md` - Guía completa de voz de marca
- `mode-examples.md` - Ejemplos de cada modo de comunicación
- `audience-adaptation.md` - Adaptación por segmento de edad
- `agent-voices.md` - Personalidad de voz de cada agente
- `checklist.md` - Checklist de validación pre-publicación

## Uso Rápido

### Para escribir copy:
1. Identifica la audiencia (30-45 o 45-60)
2. Selecciona el modo apropiado (A, B, o C)
3. Aplica la fórmula CONFRONTA → FUNDAMENTA → RESUELVE si es conversión
4. Verifica contra checklist antes de finalizar

### Para validar copy existente:
1. Carga el checklist
2. Verifica cada punto
3. Identifica correcciones necesarias
