# Detailed Brand Workflows

Complete step-by-step workflows for brand guideline creation, auditing, and application.

## Workflow 1: Creating Comprehensive Brand Guidelines

### Phase 1: Brand Foundation (Discovery)

**Step 1: Define Brand Strategy**
```
1. Brand purpose: Why does the brand exist?
2. Brand vision: Where is the brand going?
3. Brand mission: What does the brand do?
4. Brand values: What does the brand stand for?
5. Target audience: Who is the brand for?
6. Brand personality: How does the brand behave?
7. Competitive positioning: What makes the brand different?
```

**Questions to Answer:**
- What problem does our brand solve?
- What emotions should our brand evoke?
- If our brand were a person, how would you describe them?
- What are our brand's core values (choose 3-5)?
- Who are our primary and secondary audiences?
- What is our unique value proposition?

**Step 2: Conduct Brand Audit**
```
1. Inventory all existing brand materials
   - Logos and variations
   - Color usage across materials
   - Typography choices
   - Imagery styles
   - Marketing collateral
   - Digital presence

2. Assess consistency
   - Where is branding consistent?
   - Where are there discrepancies?
   - What's working well?
   - What needs improvement?

3. Competitor analysis
   - How do competitors present their brands?
   - What visual territory can we own?
   - Where can we differentiate?
```

### Phase 2: Visual Identity Development

**Step 3: Logo Design/Refinement**
```
1. Create or refine primary logo
   - Ensure scalability
   - Test in various sizes
   - Consider different contexts

2. Develop logo variations
   - Horizontal lockup
   - Vertical lockup
   - Icon/symbol only
   - Monochrome versions
   - Reversed versions

3. Define logo specifications
   - Clear space rules
   - Minimum sizes
   - Color variations
   - Incorrect usage examples

4. Create logo files
   - Vector formats (AI, EPS, SVG)
   - Raster formats (PNG with transparency)
   - Multiple color versions
   - Various size options
```

**Step 4: Color Palette Development**
```
1. Choose primary brand colors (1-3)
   - Consider color psychology
   - Test across different media
   - Ensure distinctiveness

2. Select secondary colors (2-5)
   - Complement primary palette
   - Provide flexibility
   - Support various applications

3. Define accent colors (1-3)
   - High-contrast options
   - For CTAs and highlights
   - Use sparingly

4. Establish neutral palette
   - Multiple gray tones
   - True black and white
   - Background options

5. Specify all color values
   - Pantone (for brand colors)
   - CMYK (for print)
   - RGB and HEX (for digital)
   - HSL (alternative web)

6. Test accessibility
   - Check all text/background combinations
   - Ensure WCAG AA minimum (4.5:1)
   - Aim for AAA when possible (7:1)
   - Document which combinations are approved
```

**Step 5: Typography System**
```
1. Select primary typeface
   - Versatile weights
   - Good readability
   - Professional appearance
   - Licensing for all uses

2. Choose secondary typeface (if needed)
   - Complements primary
   - Distinct purpose
   - Good pairing

3. Define type hierarchy
   - Heading levels (H1-H6)
   - Body text variations
   - Specialty text styles
   - Size scale

4. Create typography rules
   - Line heights
   - Letter spacing
   - Text alignment
   - Spacing between elements

5. Specify font files and licensing
   - Provide font files
   - Document licenses
   - List fallback fonts
   - Web font implementation
```

**Step 6: Supporting Visual Elements**
```
1. Icon style
   - Line or filled
   - Stroke weight
   - Corner radius
   - Grid system
   - Create core icon set

2. Photography style
   - Composition rules
   - Lighting preferences
   - Color treatment
   - Subject matter
   - Examples (good/bad)

3. Illustration style (if applicable)
   - Visual characteristics
   - Color usage
   - Level of detail
   - Examples

4. Patterns and textures
   - Background patterns
   - Decorative elements
   - Usage guidelines

5. Graphic elements
   - Dividers
   - Borders
   - Shapes
   - Decorative accents
```

### Phase 3: Voice and Messaging

**Step 7: Brand Voice Definition**
```
1. Define voice characteristics (4-6 traits)
   - Choose descriptive adjectives
   - Explain what each means
   - Provide examples

2. Create voice spectrum positioning
   - Formal ←→ Casual
   - Serious ←→ Playful
   - Respectful ←→ Irreverent
   - Enthusiastic ←→ Matter-of-fact
   - Mark position on each spectrum

3. Tone adaptation guidelines
   - Customer service tone
   - Marketing tone
   - Technical documentation tone
   - Social media tone
   - Internal communications tone

4. Writing style guidelines
   - Grammar rules
   - Punctuation preferences
   - Formatting standards
   - Words to use/avoid
```

**Step 8: Messaging Framework**
```
1. Craft brand positioning statement
   - Target audience
   - Category
   - Key benefit
   - Point of difference

2. Develop key messages
   - Primary message (main value prop)
   - 3-5 secondary messages
   - Proof points for each
   - Message hierarchy

3. Create message matrix
   - Messages by audience segment
   - Messages by product/service
   - Messages by communication goal

4. Write tagline/slogan (if needed)
   - Memorable
   - Meaningful
   - Aligned with positioning
   - Test with audiences
```

### Phase 4: Application Guidelines

**Step 9: Create Application Examples**
```
1. Digital applications
   - Website mockups
   - Email templates
   - Social media examples
   - Digital ads

2. Print applications
   - Business cards
   - Letterhead
   - Brochures
   - Advertisement examples

3. Product/packaging (if applicable)
   - Package design
   - Labels
   - Product tags

4. Environmental graphics
   - Signage
   - Banners
   - Trade show materials

5. Branded merchandise
   - Apparel
   - Promotional items
   - Gift packaging
```

**Step 10: Document Do's and Don'ts**
```
For each brand element, create:

1. Correct usage examples
   - Ideal applications
   - Best practices
   - Approved variations

2. Incorrect usage examples
   - Common mistakes
   - What not to do
   - Why it's wrong

3. Visual examples
   - Show correct vs. incorrect
   - Mark with ✓ and ✗
   - Brief explanations
```

### Phase 5: Document Creation

**Step 11: Structure the Guidelines Document**
```
1. Cover page
   - Brand name
   - "Brand Guidelines" or "Brand Style Guide"
   - Version number and date
   - Beautiful brand-aligned design

2. Introduction
   - Welcome message
   - Purpose of guidelines
   - How to use this document
   - Who to contact with questions

3. Brand foundation
   - Brand story
   - Mission, vision, values
   - Brand personality
   - Target audience

4. Logo guidelines
   - Logo variations
   - Clear space and sizing
   - Color variations
   - Incorrect usage

5. Color palette
   - Primary colors
   - Secondary colors
   - Accent colors
   - Neutral colors
   - Accessibility notes

6. Typography
   - Typeface families
   - Type hierarchy
   - Spacing and formatting
   - Examples

7. Imagery
   - Photography style
   - Iconography
   - Illustration (if applicable)
   - Image treatments

8. Brand voice
   - Voice characteristics
   - Tone guidelines
   - Writing style
   - Examples

9. Applications
   - Digital guidelines
   - Print guidelines
   - Environmental graphics
   - Do's and don'ts

10. Templates and resources
    - Where to find templates
    - Asset library location
    - Contact information
    - Approval process

11. Appendix
    - Detailed specifications
    - File naming conventions
    - Vendor guidelines
    - Legal disclaimers
```

**Step 12: Design the Guidelines Document**
```
1. Apply brand to the guidelines itself
   - Use brand colors, typography, imagery
   - The document should exemplify the brand
   - Make it visually engaging

2. Ensure usability
   - Clear navigation/table of contents
   - Logical organization
   - Searchable PDF
   - Bookmarks for easy jumping

3. Make it accessible
   - High-contrast text
   - Alt text for images
   - Logical reading order
   - Accessible PDF features

4. Create different versions
   - Full comprehensive version (50-100 pages)
   - Quick reference guide (10-15 pages)
   - Digital-specific guide (if needed)
   - Print-specific guide (if needed)
```

### Phase 6: Distribution and Implementation

**Step 13: Create Digital Asset Library**
```
1. Organize brand assets
   /Brand_Assets
   ├── /Logos
   │   ├── /Primary
   │   ├── /Secondary
   │   ├── /Icon
   │   └── /Variations
   ├── /Colors
   │   └── color-palette.ase (Adobe Swatch Exchange)
   ├── /Fonts
   │   ├── /Primary_Typeface
   │   └── /Secondary_Typeface
   ├── /Images
   │   ├── /Photography
   │   ├── /Icons
   │   └── /Patterns
   ├── /Templates
   │   ├── /Documents
   │   ├── /Presentations
   │   ├── /Email
   │   └── /Social_Media
   └── /Guidelines
       ├── Brand_Guidelines_Full.pdf
       └── Brand_Guidelines_Quick_Reference.pdf

2. Set up access and permissions
   - Cloud storage (Google Drive, Dropbox, etc.)
   - Access levels (view, download, edit)
   - Version control
   - Update notifications

3. Create asset request process
   - How to request new assets
   - Approval workflow
   - Turnaround expectations
```

**Step 14: Training and Rollout**
```
1. Create training materials
   - Presentation deck explaining guidelines
   - Video walkthrough
   - FAQ document
   - Quick tips sheet

2. Conduct training sessions
   - Marketing team (deep dive)
   - Leadership team (overview)
   - All employees (brand basics)
   - External partners/vendors (relevant sections)

3. Gather feedback
   - Is anything unclear?
   - Are there missing scenarios?
   - What additional resources are needed?
   - Revise guidelines based on feedback

4. Announce rollout
   - Internal announcement
   - Timeline for implementation
   - Resources available
   - Support contacts
```

**Step 15: Maintenance and Updates**
```
1. Establish governance
   - Who owns the brand guidelines?
   - Who can approve changes?
   - Update schedule (annual review minimum)

2. Track issues and questions
   - Maintain FAQ based on real questions
   - Document edge cases
   - Identify gaps in guidelines

3. Plan for evolution
   - Brand refresh vs. rebrand
   - When to update guidelines
   - Version control and archiving
   - Communication of changes

4. Monitor compliance
   - Regular brand audits
   - Feedback on implementations
   - Recognition for good brand stewardship
   - Gentle correction when needed
```

## Workflow 2: Auditing Materials for Brand Compliance

### Audit Preparation

**Step 1: Define Audit Scope**
```
1. What materials will you audit?
   □ Website and digital properties
   □ Marketing collateral (brochures, flyers, etc.)
   □ Email templates and campaigns
   □ Social media profiles and posts
   □ Presentations
   □ Print materials (business cards, signage, etc.)
   □ Product packaging
   □ Internal documents
   □ Advertising (digital and print)
   □ Video and multimedia

2. What time period?
   - Current materials only
   - Last 6 months
   - Last year
   - All materials in circulation

3. Who is responsible for each area?
   - Assign audit leads
   - Set deadlines
   - Determine review process
```

**Step 2: Create Audit Checklist**
```
Develop checklist based on brand guidelines:

LOGO USAGE
□ Correct logo version used
□ Logo meets minimum size requirements
□ Proper clear space maintained
□ Logo not distorted or altered
□ Correct logo color (not custom colors)
□ Logo placement consistent with guidelines
□ No unauthorized effects on logo

COLOR USAGE
□ Only approved brand colors used
□ Correct color values (not approximations)
□ Appropriate color combination
□ Sufficient contrast for accessibility
□ Consistent color usage across materials

TYPOGRAPHY
□ Approved fonts used
□ Correct font weights and styles
□ Proper hierarchy maintained
□ Appropriate sizes and spacing
□ No unauthorized font substitutions

IMAGERY
□ Images match brand photography style
□ Icons consistent with brand style
□ Appropriate image quality
□ Proper image licensing/permissions
□ Images support brand messaging

MESSAGING
□ Brand voice evident
□ Appropriate tone for context
□ Key messages present and accurate
□ No off-brand language
□ Grammar and style guide followed

LAYOUT/COMPOSITION
□ Follows brand spacing guidelines
□ Grid system used appropriately
□ Visual hierarchy clear
□ Branded elements positioned correctly
□ Overall composition feels on-brand

ACCESSIBILITY
□ Color contrast meets WCAG standards
□ Text is readable
□ Alt text present (digital)
□ Logical reading order
□ Accessible formatting used
```

### Conducting the Audit

**Step 3: Systematically Review Materials**
```
For each piece of material:

1. Document basic information
   - Material name/title
   - Type (brochure, webpage, etc.)
   - Creation date
   - Department/owner
   - Current status (active, archived)

2. Evaluate against checklist
   - Check each criterion
   - Note compliant items (✓)
   - Flag non-compliant items (✗)
   - Mark items needing clarification (?)

3. Capture issues with details
   - Screenshot or photo of issue
   - Description of what's wrong
   - Reference to guideline violated
   - Severity: Critical / Moderate / Minor

4. Rate overall compliance
   - Fully compliant (90-100%)
   - Mostly compliant (70-89%)
   - Partially compliant (50-69%)
   - Non-compliant (<50%)
```

**Step 4: Categorize Issues**
```
Group findings by:

By Type:
- Logo issues
- Color issues
- Typography issues
- Imagery issues
- Messaging issues
- Layout issues
- Accessibility issues

By Severity:
- Critical: Brand misrepresentation, legal issues
- Moderate: Noticeable inconsistencies
- Minor: Small deviations, polish items

By Department:
- Marketing
- Sales
- Product
- HR
- Executive
- External (vendors, partners)

By Material Type:
- Digital
- Print
- Environmental
- Product/packaging
```

### Analysis and Reporting

**Step 5: Analyze Patterns**
```
Look for:

1. Common issues
   - What violations appear most frequently?
   - Are certain guidelines unclear or ignored?
   - Do specific departments struggle more?

2. Root causes
   - Lack of awareness of guidelines?
   - No access to approved assets?
   - Guidelines unclear or incomplete?
   - No approval process in place?
   - Vendor not following guidelines?
   - Outdated templates still in use?

3. Positive examples
   - What materials are exemplary?
   - Which teams are doing it right?
   - What can others learn from them?
```

**Step 6: Create Audit Report**
```
Report Structure:

1. Executive Summary
   - Overall compliance rate
   - Key findings
   - High-priority issues
   - Recommendations

2. Methodology
   - Scope of audit
   - Materials reviewed
   - Criteria used
   - Time period

3. Detailed Findings
   - Compliance by category
   - Issue frequency charts
   - Examples of violations
   - Department-by-department breakdown

4. Specific Issues
   - Critical issues requiring immediate action
   - Moderate issues for near-term addressing
   - Minor issues for backlog

5. Best Practices
   - Highlight excellent examples
   - What's working well
   - Teams to recognize

6. Recommendations
   - Prioritized action items
   - Process improvements
   - Training needs
   - Guideline updates needed
   - Resource requirements

7. Action Plan
   - Immediate fixes (0-30 days)
   - Short-term updates (1-3 months)
   - Long-term improvements (3-12 months)
   - Responsible parties
   - Success metrics
```

### Remediation

**Step 7: Prioritize Corrections**
```
Prioritization Matrix:

HIGH PRIORITY:
- Customer-facing materials with critical issues
- High-visibility materials (website homepage, etc.)
- Legal or compliance risks
- Brand reputation risks

MEDIUM PRIORITY:
- Moderate issues in customer-facing materials
- Internal materials with critical issues
- Frequently used templates

LOW PRIORITY:
- Minor issues in any materials
- Low-visibility or rarely used materials
- One-off historical materials being phased out
```

**Step 8: Execute Corrections**
```
1. Immediate fixes (Quick wins)
   - Update website with correct logo
   - Fix email signature templates
   - Update social media profile images
   - Correct obvious color/font issues

2. Short-term updates
   - Redesign key marketing materials
   - Update presentation templates
   - Refresh business cards
   - Revise email templates

3. Long-term projects
   - Website redesign
   - Complete collateral refresh
   - Packaging redesign
   - Environmental graphics update

4. Process improvements
   - Implement approval process
   - Create better templates
   - Set up asset management system
   - Schedule regular audits
```

**Step 9: Prevent Future Issues**
```
1. Improve access to guidelines
   - Make guidelines easily findable
   - Create quick reference guides
   - Provide training

2. Create better tools
   - Locked templates (protect brand elements)
   - Pre-approved social media templates
   - Brand asset library
   - Design request system

3. Establish governance
   - Brand approval process
   - Regular review checkpoints
   - Designated brand champions
   - Vendor management guidelines

4. Schedule ongoing monitoring
   - Monthly spot checks
   - Quarterly mini-audits
   - Annual comprehensive audit
   - Real-time monitoring (website, social media)
```

## Workflow 3: Applying Brand to New Documents

### Planning Phase

**Step 1: Understand the Document Purpose**
```
Ask:
1. What is the document type?
   - Report, presentation, brochure, email, etc.
   - Determines which guidelines to apply

2. Who is the audience?
   - External (customers, prospects, partners)
   - Internal (employees, leadership)
   - Affects tone and formality level

3. What is the goal?
   - Inform, persuade, educate, sell
   - Influences messaging and design choices

4. Where will it be used?
   - Print or digital or both
   - Affects technical specifications

5. What is the timeline?
   - Rush job or ample time
   - Affects thoroughness of brand application
```

**Step 2: Gather Brand Resources**
```
Collect:
1. Brand guidelines document
   - Review relevant sections
   - Note any specific rules for this document type

2. Templates (if available)
   - Check for existing template for this document type
   - Use as starting point if appropriate

3. Brand assets
   - Logo files (correct version and format)
   - Color values
   - Font files
   - Icon set
   - Approved images

4. Examples
   - Look at similar past documents
   - Identify what worked well
   - Note approved patterns
```

### Design Phase

**Step 3: Set Up Document Structure**
```
1. Choose appropriate template or create structure
   - Use locked brand template if available
   - Set up master pages/layouts
   - Establish grid system

2. Apply brand foundations
   - Set color palette in document
   - Load brand fonts
   - Import logo and brand elements
   - Set up styles (heading styles, paragraph styles)

3. Establish hierarchy
   - Define heading levels needed
   - Set up body text styles
   - Create emphasis styles (bold, italic, color)
   - Define list styles

4. Set up spacing
   - Margins (per guidelines)
   - Internal spacing between elements
   - Consistent padding
   - Grid alignment
```

**Step 4: Apply Visual Brand Elements**
```
1. Logo placement
   - Position per guidelines
   - Use correct logo variation
   - Ensure minimum size
   - Maintain clear space

2. Color application
   - Use exact brand colors (not approximations)
   - Apply color hierarchy:
     * Primary color: Headers, key elements
     * Secondary colors: Supporting elements
     * Accent color: CTAs, highlights
     * Neutrals: Body text, backgrounds
   - Check color contrast
   - Test for color accessibility

3. Typography
   - Apply brand fonts
   - Use type hierarchy from guidelines
   - Set appropriate sizes
   - Adjust line-height and spacing
   - Ensure readability

4. Imagery
   - Select on-brand images
   - Apply consistent image treatment
   - Ensure proper quality/resolution
   - Use brand-style icons
   - Maintain consistent visual style

5. Graphic elements
   - Use brand patterns/textures if applicable
   - Apply brand-style dividers, borders
   - Consistent graphic treatment
```

### Content Phase

**Step 5: Write On-Brand Content**
```
1. Apply brand voice
   - Review voice characteristics
   - Write in appropriate tone for audience
   - Use brand vocabulary
   - Avoid off-brand language

2. Structure content
   - Clear hierarchy
   - Scannable format
   - Logical flow
   - Appropriate length

3. Incorporate key messages
   - Lead with main message
   - Support with secondary messages
   - Include proof points
   - Reinforce brand positioning

4. Write headlines
   - Clear and benefit-focused
   - Consistent style (sentence case, etc.)
   - Appropriate length
   - Aligned with brand voice

5. Create calls-to-action
   - Action-oriented language
   - Specific and clear
   - Appropriately placed
   - Brand-aligned phrasing
```

### Review and Refinement

**Step 6: Self-Review Against Guidelines**
```
Go through brand checklist:

LOGO
□ Correct version and color
□ Proper size and placement
□ Clear space maintained
□ Not distorted

COLOR
□ Only brand colors used
□ Correct color values
□ Appropriate combinations
□ Accessible contrast

TYPOGRAPHY
□ Brand fonts only
□ Correct hierarchy
□ Proper sizing and spacing
□ Readable and accessible

IMAGERY
□ On-brand style
□ Appropriate quality
□ Properly sized and placed
□ Licensed/permitted

MESSAGING
□ Brand voice evident
□ Appropriate tone
□ Key messages present
□ No off-brand language

LAYOUT
□ Follows grid system
□ Consistent spacing
□ Clear visual hierarchy
□ Professional appearance

OVERALL
□ Feels on-brand
□ Achieves document goals
□ Error-free
□ Ready for review
```

**Step 7: Get Feedback and Approval**
```
1. Internal review
   - Share with colleagues
   - Get content feedback
   - Check for errors
   - Revise as needed

2. Brand approval (if required)
   - Submit to brand team/owner
   - Address any brand concerns
   - Make required changes
   - Get final sign-off

3. Stakeholder approval
   - Share with document owner
   - Incorporate feedback
   - Get final approval

4. Legal/compliance review (if needed)
   - Ensure regulatory compliance
   - Check claims and statements
   - Verify required disclosures
```

### Finalization

**Step 8: Prepare Final Files**
```
1. Proofread thoroughly
   - Spelling and grammar
   - Formatting consistency
   - Broken links (if digital)
   - Image quality

2. Check technical specs
   - Correct dimensions
   - Proper resolution (300 DPI print, 72 DPI web)
   - Color mode (CMYK print, RGB digital)
   - Bleed and trim marks (if print)

3. Create final versions
   - Working file (editable source)
   - Print PDF (high-res, CMYK, with bleeds)
   - Digital PDF (optimized file size, RGB)
   - Other formats as needed (PPT, DOCX, etc.)

4. Organize files
   - Clear file naming
   - Version numbers
   - Organized folder structure
   - Archive source files

5. Prepare for distribution
   - Upload to appropriate location
   - Set permissions
   - Create shortened links (if needed)
   - Provide to stakeholders
```

**Step 9: Document and Archive**
```
1. Add to brand asset library
   - Save in appropriate folder
   - Tag for searchability
   - Update asset inventory

2. Document details
   - Purpose and usage
   - Audience
   - Distribution channels
   - Approval date and approvers

3. Create reusable template (if applicable)
   - Save as template
   - Lock brand elements
   - Add instructions
   - Make available to team

4. Note learnings
   - What worked well
   - Challenges encountered
   - Process improvements
   - Update guidelines if needed
```

## Workflow 4: Managing Multiple Brands/Sub-brands

### Brand Architecture Setup

**Step 1: Define Brand Relationship**
```
Determine architecture type:

1. Branded House (Single brand)
   - All offerings under one brand
   - Example: Apple (iPhone, iPad, Mac, etc.)
   - Approach: One set of guidelines applied consistently

2. House of Brands (Multiple distinct brands)
   - Each brand stands alone
   - Example: Procter & Gamble (Tide, Gillette, etc.)
   - Approach: Separate guidelines for each brand

3. Endorsed Brands (Sub-brands with parent endorsement)
   - Sub-brands with parent brand support
   - Example: Marriott (Courtyard by Marriott, etc.)
   - Approach: Parent guidelines + sub-brand variations

4. Hybrid (Combination)
   - Mix of approaches
   - Complex organizations
   - Approach: Tiered guideline system
```

**Step 2: Establish Brand Hierarchy**
```
For endorsed or hybrid structures:

1. Define brand levels
   - Corporate/parent brand
   - Division/business unit brands
   - Product/service brands
   - Campaign/initiative brands

2. Determine relationships
   - Which brands endorse which
   - Co-branding rules
   - Prominence hierarchy
   - Lock-up specifications

3. Create usage rules
   - When to use parent brand alone
   - When to use sub-brand alone
   - When to co-brand
   - Decision tree for brand selection
```

### Multi-Brand Guidelines Development

**Step 3: Create Tiered Guidelines**
```
Level 1: Master Brand Guidelines
- Overarching brand principles
- Corporate identity standards
- Shared elements across all brands
- Governance and approval processes

Level 2: Sub-Brand Guidelines
- How sub-brand relates to master brand
- Sub-brand specific elements
- When sub-brand can stand alone
- Co-branding specifications

Level 3: Application Guidelines
- Specific use cases
- Channel-specific rules
- Regional adaptations
- Special circumstances
```

**Step 4: Define Shared vs. Unique Elements**
```
SHARED ELEMENTS (Consistent across brands):
- Corporate logo (for endorsement)
- Core color palette (or color family)
- Typography system (or font family)
- Brand voice foundations
- Messaging architecture
- Quality standards

UNIQUE ELEMENTS (Differ by sub-brand):
- Sub-brand logo/mark
- Extended color palette
- Imagery style variations
- Tone adaptations
- Target audience specifics
- Product-specific messaging

DOCUMENT FOR EACH:
- What is shared/required
- What can be adapted/customized
- Approval needed for variations
- Examples of correct application
```

**Step 6: Implement Brand Management Tools**
