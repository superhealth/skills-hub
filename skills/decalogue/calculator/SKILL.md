# Calculator Skill

## 功能描述

Calculator Skill 提供数学计算能力，支持基本的四则运算、幂运算等。

## 使用方法

### 基本运算

支持以下运算符：
- `+` 加法
- `-` 减法
- `*` 乘法
- `/` 除法
- `**` 或 `^` 幂运算
- `()` 括号

### 示例

```
2 + 3 * 4 = 14
10 ** 2 = 100
(5 + 3) * 2 = 16
```

## 注意事项

1. 表达式必须是安全的数学表达式
2. 不支持变量和函数调用
3. 除零会返回错误
