# Post-Show Follow-up

创建分层展后跟进邮件序列，将展位交流转化为销售机会。

## 功能

告诉 Claude 刚结束的展会名称和你的产品，它会生成：

- 3 级线索分类（热 / 温 / 冷）
- 每级多封邮件序列，每封邮件采用不同切入角度
- 个性化字段和 CRM 合并标签支持
- 推荐发送时间和实用技巧

## 使用方式

```
MEDICA 刚结束，帮我写跟进邮件给收集到的线索。
```

```
我们在 Interpack 扫了 200 个 badge，其中 30 个有实质对话，10 个问了价格。帮我写各组的跟进邮件。
```

```
帮我写一封展后感谢邮件，给在 CES 参观过我们 demo 的人。
```

## 安装

```bash
claude install-skill /path/to/trade-show-skills/post-show-followup
```
